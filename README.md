# sitemap_robot
基于网站sitemap 抓取网页生成静态HTML

由于时间关系，目前仅支持根目录和一级目录源码抓取。

不支持图片，样式抓取，仅支持源码抓取。

已实现功能：

    支持多sitemap 抓取，不是同时

    支持追加数据抓取，见代码36行
